# Aptos Indexer Expert

Expertise in querying and processing Aptos blockchain data using the Indexer.

## When to Use

- Querying complex data (NFTs, coin balances, transaction history) via GraphQL
- Designing custom indexer processors for specific dApp needs
- Understanding the Indexer architecture (GRPC, Database, Hasura)
- Optimizing data retrieval for high-performance frontends

## Querying with GraphQL

Aptos provides a GraphQL API for indexing data.
- **Endpoint:** `https://api.mainnet.aptoslabs.com/v1/graphql`
- **Common Queries:** Fetching account transactions, token metadata, and activity.
