# skill-master (AI Agent Skill Master)

The ultimate authority on AI Agent Skills. Born for the SKILLS ecosystem to analyze, architect, and author high-performance skills.

## Capabilities
- `analyze`: Evaluate an existing skill's clarity, actionability, and structural integrity.
- `architect`: Design custom extensions and hooks for agentic workflows.
- `author`: Create senior-engineered `SKILL.md` documents following the industry-leading standards (like Anthropic & OilOil guides).

## The Anatomy of a Perfect Skill
1. **Metadata**: Clear name, concise description, and precise domain scoping.
2. **Workflow**: Step-by-step instructions for the agent to follow (the "how-to").
3. **Core Principles**: Behavioral constraints and high-level logic (the "why").
4. **Checklists**: Verification logic to ensure the task is completed correctly (the "quality").

## Design Philosophy
- **Task-First**: Every line must help the agent complete a specific goal.
- **Actionable & Direct**: Use imperative verbs. Avoid conversational filler or general advice.
- **Progressive Disclosure**: Keep core rules at the top; hide edge cases in references.
- **System-Aware**: Understand how tools and skills interact within a complex workspace.

## Workflow: `authoring` a new skill
1. **Identify the Gap**: What can't the agent currently do with high precision?
2. **Scoping**: Define the input (context) and output (result).
3. **Draft Principles**: Set the non-negotiables (hard rules).
4. **Refine Workflow**: Create a logical, branching sequence of steps.
5. **Quality Check**: Run the `skill-master` review pass on the draft.
