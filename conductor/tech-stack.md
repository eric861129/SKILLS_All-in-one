# Tech Stack

## 1. 前端 (Frontend)
- **Framework:** **React 19 (Vite 7)** — 提供高性能的開發體驗與現代 React 特性。
- **Language:** **TypeScript** — 確保代碼的類型安全與可維護性。
- **Styling:** **Tailwind CSS 4** — 遵循「反 AI Slop」設計規範，實現精細的 UI 控制。
- **Routing:** **React Router DOM** — 處理 SPA 路由與 URL 參數同步。
- **Icons:** **Lucide Icons** — 專業的向量圖標庫，嚴禁使用 Emoji。

## 2. 工具與函式庫 (Tools & Libraries)
- **Search:** **Fuse.js** — 輕量化前端模糊搜尋，支援多維度加權檢索。
- **Compression:** **JSZip** — 實現多檔案打包與自動化下載邏輯。

## 3. 後端與數據 (Backend & Data)
- **Backend Bridge:** **Cloudflare Workers** — 處理 CORS、API 路由與數據轉發。
- **Database:** **Turso (SQLite)** — 分佈式邊緣數據庫，儲存技能元資料與下載計數。

## 4. 託管與部署 (Hosting & Deployment)
- **Hosting:** **GitHub Pages** — 靜態站點託管。
- **CI/CD:** **GitHub Actions** — 自動化建置、 manifest 生成與部署流程。
- **Domain:** 自定義域名支援與 SPA 重導向配置。
